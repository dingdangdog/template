# Template-Vuetify-Home

这是一个简单的首页前端模板项目，如需开发官网首页，可以取之即用。

Demo: https://home.template.oldmoon.top/

## 页面截图

![home](./images/home.jpg)

## 主要依赖

- Vue3
- TypeScript
- Pinia
- Vuetify
- Axios
- Vite
- Vitest
- vite-plugin-vue-devtools：默认关闭，可自行开启

## 常用指令

推荐使用 `pnpm` 安装依赖：

```sh
# 安装依赖
pnpm i
```

```sh
# 开发运行
npm run dev
```

```sh
# 类型检查并编译
npm run build
```
